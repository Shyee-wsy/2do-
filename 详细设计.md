# 详细设计  
### 一、建表设计  
1. listConnections  
| 字段 | 类型 | 描述 |  
--- --- --- ---
| id | Int | 清单Id |
| listName | string | 清单名称，除了默认的Todo清单，其他为用户自定义清单 |  
| isDefault | Boolean | 标识是否为默认清单（Todo） |  
| nodes | items | 所有当前清单的item 集合|
2. item   
| 字段 | 类型 | 描述 |  
| id | Int | 标识每个item |  
| content | string | item的内容 |
| listName | string | 隶属于的清单名称 | 
| isFinished | Boolean | 是否被标记为完成 |
